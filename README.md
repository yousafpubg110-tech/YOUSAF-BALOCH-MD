## 📦 Manual Installation

### Prerequisites

- Node.js v18 or higher
- Git
- MongoDB (optional, for persistent storage)

### Local Setup

```bash
# Clone the repository

# Navigate to directory
cd YOUSAF-BALOCH-MD

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your details

# Start the bot
npm start
```
